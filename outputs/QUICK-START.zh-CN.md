# 公会试炼插件最简教程

## 一、安装

1. 浏览器先安装 Tampermonkey（油猴）。
2. 会长安装[会长端脚本](https://raw.githubusercontent.com/overjjjj/-mwi-guild-trial-data/main/outputs/milkyway-guild-trial-allocator.user.js)。
3. 普通会员安装[会员端脚本](https://raw.githubusercontent.com/overjjjj/-mwi-guild-trial-data/main/outputs/milkyway-guild-trial-member.user.js)。
4. 安装后刷新游戏，右下角会出现插件按钮。

## 二、会员怎么用

首次使用：

1. 打开右下角“公会试炼会员端”。
2. 展开“连接设置”，填写会长发来的公会编号。
3. 点击“保存连接”。
4. 点击“一键上传并查看分配”。

以后装备、技能或等级变化时，再点一次“一键上传并查看分配”即可。

会长发布方案后，点击“只查看最新分配”，再按显示的方向去游戏中手动报名。

## 三、会长首次设置

1. 打开右下角“公会试炼分配助手”。
2. 进入“设置”，点击“创建在线公会”。
3. 点击“复制公会编号”，把编号发给所有会员。
4. 点击“复制管理备份”，保存好备份，不要发给普通会员。

## 四、会长每周怎么用

按下面顺序操作：

1. 点击“读取试炼”，插件会自动进入公会试炼页面。
2. 打开游戏的公会成员页面，点击“读取成员”。
3. 点击“同步本周名单”。
4. 通知会员点击“一键上传并查看分配”。
5. 点击“拉取会员资料”。
6. 给坦克、自然治疗、减益成员设置职责或固定战斗位置。
7. 点击“生成分配”，需要时再点击“模拟战斗”。
8. 确认无误后点击“发布方案”。

顶部蓝色按钮会提示下一步，正常使用时跟着它点击即可。

## 五、常见提示

- “公会名单未同步”：会长重新执行“读取成员”和“同步本周名单”。
- “试炼数据不完整”：重新点击“读取试炼”，确认显示生活 `4/4`、战斗 `2/2`。
- “缺少完整配装”：让提示中的会员重新上传角色资料。
- 插件只负责推荐分配，不会自动替会员报名。
